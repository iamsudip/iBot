iBot
====

iBot, an IRC bot with human-brain

How to run
----------

To run simply do::

	$ python iBot.py <nickserv password> <channel1> <channel2>

channel name must be without '#'. Ex. To join #dgplug use only 'dgplug' as::

	$ python iBot.py 12345pass dgplug hasgeek
